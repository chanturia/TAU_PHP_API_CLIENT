## Installation & Usage
### Composer

To install the bindings via [Composer](http://getcomposer.org/), add the following to `composer.json`:

```
{
  "repositories": [
    {
      "type": "git",
      "url": "https://github.com/vaspapa7/tau-api-v1.git"
    }
  ],
  "require": {
    "vaspapa7/tau-api-v1": "*@dev"
  }
}
```

Then run `composer install`
